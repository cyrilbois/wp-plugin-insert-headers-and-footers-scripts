Insert Headers And Footers Scripts allows you to insert scripts, CSS or HTML in the header and footer of your website.

== Description ==

A simple plugin to insert scripts, CSS or HTML in the header and footer of your website.

= Summary of Insert Headers And Footers Scripts features =

* Insert scripts
* Insert CSS
* Insert HTML
* Insert into the header 
* Insert into the the footer (above the closing </body> tag)
* Insert in entire site
* Insert on specific posts or pages

== Installation ==

1. Unzip the downloaded 'insert-headers-and-footers-scripts' zip file
2. Upload the 'insert-headers-and-footers-scripts' folder into the 'wp-content/plugins/' directory/
3. Activate Insert Headers And Footers Scripts through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I use this plugin to install Google AdSense? =

Yes! All you need to do is insert the Google AdSense script (for Auto ads) in the "Scripts in Header (in the head section)" field.

= Can I use this plugin to install Google Analytics? =

Yes! All you need to do is insert the Google Analytics script in the "Scripts in Header (in the head section)" field.

= How to insert headers or footers scripts on a specific post? =

You just need to go to the bottom of the post screen and insert your code in the header or footer fields.
